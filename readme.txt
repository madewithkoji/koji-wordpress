== Description ==

Easily embed <a href="https://withkoji.com/" target="_blank">Koji Apps</a> in your Wordpress Gutenberg posts and pages. Koji is the app store for the creator economy, and has over 200 web-based mini-apps that support all kinds of functionality, including e-commerce apps, games, and other types of powerful interactive experiences.

This plugin brings all of Koji's apps to Gutenberg, letting you easily add things like the ability to sell downloadable files, sell custom video requests like shoutouts and birthday greetings, embed games that can be played by visitos, and more. Koji apps are free to use. Some apps take a transaction fee when users make a purchase.

= Plugin Features =
* Embed any Koji app
* Easily create new apps from the Koji App Store
* Over 200 apps, with new apps being added every day
* Frictionless user-experience, no app installs required
* No Koji Account required to use apps (account is required to create new apps)

== Installation ==

Install Koji Block for Gutenberg either via the WordPress plugin directory or by uploading the files to your server at wp-content/plugins.

== Frequently Asked Questions ==

= Is a Koji Account required? =

Visitors do not need a Koji account to use your apps. You will need a free Koji account to create new apps.

= Do users need to install anything? =

No, Koji apps are "web apps," meaning they run instantly inside every browser.

= Is Koji free to use? =

Koji apps are free to create and to use. If you use apps with ecommerce capabilities, there is a fee for each transaction. You can find the fee percentage on the app page when creating the app.
